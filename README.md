
عداد الأذكار الإلكتروني – Athkar Counter PWA  
تطبيق ويب تقدمي بسيط، أنيق وخفيف جدًا للتسبيح والمحافظة على أذكار الصباح والمساء يوميًا

#### نبذة عن المشروع
«عداد الأذكار» هو تطبيق ويب تقدمي (PWA) تفاعلي باللمس، مصمم بحب ليكون رفيقك اليومي في التسبيح وأذكار الصباح والمساء.  
يعمل 100% بدون إنترنت، خفيف جدًا (أقل من 100 كيلوبايت)، سريع الاستجابة، ويمكن تثبيته على الهاتف أو الكمبيوتر كتطبيق أصلي تمامًا.

تم تطويره بأسلوب بسيط ونقي باستخدام HTML + CSS + Vanilla JavaScript فقط، بدون أي مكتبات ثقيلة، مع التركيز على الأداء والراحة البصرية.

#### الميزات الرئيسية
- عدّاد تفاعلي باللمس أو النقر (اضغط في أي مكان على الشاشة)  
- مجموعة مختارة ومُرتبة من أذكار الصباح والمساء من السنة الصحيحة  
- وضع ليلي مريح للعين (Dark Mode) مع تبديل تلقائي حسب إعدادات الجهاز  
- تصميم أنيق وبسيط جدًا، خالٍ من الإعلانات والتتبع تمامًا  
- يعمل بدون إنترنت نهائيًا بعد الزيارة الأولى  
- قابل للتثبيت على أندرويد وآيفون وويندوز وماك ولينكس  

#### رابط التطبيق المباشر
https://smasemt8791.github.io/Athkar_counter/

#### التقنيات المستخدمة
- HTML5 + CSS3 + Vanilla JavaScript (نقي 100%)  
- Progressive Web App (PWA) كامل المواصفات  
- Service Worker + Cache API  
- Local Storage لحفظ العدادات  
- Vibration API + Haptic Feedback  
- Web Manifest + أيقونات مخصصة لكل الأجهزة  

#### للمطورين – التشغيل محليًا
لا يحتاج أي تثبيت أو بناء!  
فقط افتح الملف `index.html` في أي متصفح حديث، أو استخدم خادم بسيط:

```bash
npx serve
# أو
python -m http.server 3000
```

**Athkar Counter – Touch Tasbih & Daily Adhkar PWA**  
A beautiful, ultra-light, fully offline digital tasbih and daily remembrance app

#### Project Overview
Athkar Counter is a minimalist, touch-based Progressive Web App designed to help you maintain your daily morning and evening adhkar, and to make tasbih anywhere, anytime — completely offline.

Built with pure HTML, CSS, and Vanilla JavaScript (no frameworks, no bloat), it weighs less than 100 KB and works perfectly even without internet.

#### Key Features
- Tap-anywhere counter (just touch the screen to count)  
- Curated collection of authentic morning & evening adhkar  
- Comfortable Dark Mode (auto-detects system preference)  
- Zero ads, zero tracking, 100% private  
- Full offline functionality after first visit  
- Installable on Android, iPhone, Windows, Mac, Linux  

#### Live App
https://smasemt8791.github.io/Athkar_counter/

#### Tech Stack
- Pure HTML5 + CSS3 + Vanilla JavaScript  
- Full PWA (Service Worker, Cache, Manifest)  
- Local Storage  
- Vibration API  
- Custom icons for all platforms  

#### Run Locally
No build required! Just open `index.html` or serve it:

```bash
npx serve
```
